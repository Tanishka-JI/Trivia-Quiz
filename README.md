# 🧠 Trivia Master

🔗 [Live Demo] trivia-quiz-delta-two.vercel.app

## Features
- Category & difficulty selection
- Countdown timer per question  
- Real-time answer feedback
- Score tracking with derived state
- Built with React Context API

## Tech Stack
React, Vite, Tailwind CSS, Context API, OpenTDB API

## What I Learned
- useEffect cleanup functions to prevent timer memory leaks
- Refactoring score from manual counter to derived value
- Separating API logic (api.js) from state management (Context)

## Run Locally
git clone [https://github.com/Tanishka-JI/Trivia-Quiz]
npm install
npm run dev
